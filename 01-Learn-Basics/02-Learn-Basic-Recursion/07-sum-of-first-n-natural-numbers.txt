🟢 Problem

Sum of first N Natural Numbers

Matlab:

N = 5
Answer = 5 + 4 + 3 + 2 + 1 = 15

1️⃣ Brute Force Approach (Sochne ka Natural Tarika)
🧠 Mathematics Samjho

Natural numbers:

1, 2, 3, 4, 5 …

Toh sum ka matlab:

Add everything from 1 to N.

🔁 Loop Boundary Logic

Loop kahan tak chalega?

1 se N tak.

✅ Condition Logic

Har iteration mein current number add karna hai.

🔚 Return Logic

Final accumulated sum return karna hai.

Code (Brute Force)
def sum_n(n):
    total = 0
    
    for i in range(1, n + 1):
        total = total + i
    
    return total

Time Complexity = O(n)

❓ Thinking Question

Agar loop range(1, n) likh doon
toh kya problem hogi?


2️⃣ Using Formula (Mathematical Optimization)

Maths se pata hai:

Sum of first N numbers =

N × (N + 1) / 2

Example:

N = 5
5 × 6 / 2 = 15

Code
def sum_n(n):
    return n * (n + 1) // 2

Time Complexity = O(1)

❓ Thinking Question

Ye formula recursion se fast kyu hai?

3️⃣ Recursive Approach (Slowly Samjho)
🧠 Mathematical Thinking

sum(5) = 5 + sum(4)

sum(4) = 4 + sum(3)

sum(3) = 3 + sum(2)

sum(2) = 2 + sum(1)

sum(1) = 1 + sum(0)

sum(0) = 0

🔹 Base Case

If n == 0 → return 0

🔹 Reduction Step

n → n-1

🔹 Return Logic

return n + sum(n-1)

Code
def sum_n(n):
    if n == 0:
        return 0
    
    return n + sum_n(n - 1)
🧠 Visual Stack (n = 3)

Down phase:

3 + ?
2 + ?
1 + ?
0

Up phase:

1 + 0 = 1
2 + 1 = 3
3 + 3 = 6

Time Complexity

Calls:

3
2
1
0

Total ≈ n

Time = O(n)

🎯 Pattern Recognition

Har linear recursion problem mein dekho:

Work kahan ho raha hai? (before ya after call)
